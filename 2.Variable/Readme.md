# 변수

- 기본형식

    ```c
    타입 변수명 = 값;

    타입 변수명1 = 값

    타입 변수명2= 값2, 변수명3 = 값3


    int main()
    {
        int num1 = 10;               // 변수를 선언하면서 값 할당(초기화)
        int num2 = 20, num3 = 30;    // 변수 여러 개를 선언하면서 값 할당(초기화)

        printf("%d %d %d\n", num1, num2, num3);    // 10 20 30: 변수에 저장된 값을 %d로 출력

        return 0;
    }


    ```

- 정의와 초기화 따로

    ```c
    
    타입 변수명1, 변수명2, 변수명3;

    변수명1 = 값1
    변수명2 = 값2 
    ...

    #include <stdio.h>

    int main()
    {
        int num1 = 10;               // 변수를 선언하면서 값 할당(초기화)
        int num2 = 20, num3 = 30;    // 변수 여러 개를 선언하면서 값 할당(초기화)

        printf("%d %d %d\n", num1, num2, num3);    // 10 20 30: 변수에 저장된 값을 %d로 출력

        return 0;
    }



    ```
