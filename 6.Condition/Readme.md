# 조건문

<br>

## if, else if, else

```c

if 조건1
{
    코드1
}
else if 조건2
{
    코드2
}
else if 조건3
{
    코드3
}
else
{
    모두 포함되지 않은 때 실행될 코드
}


```

## switch case

```c

// 간단한 코드 실행 
switch (판단할 변수)
{
    case 조건1:
        코드 1
        break;
    
    case 조건2:
        코드 2
        break;
    
    .
    .
    .

    default:
        아무 case에도 해당되지 않았을 때 실행한 코드

}



```
